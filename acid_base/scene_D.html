📖 Read-through — all 13 symbols
1. All RTA = normal-AG acidosis (NH4+)
All RTAs cause a NORMAL anion-gap (hyperchloremic) metabolic acidosis from impaired acid handling. Urine ammonium / positive urine anion gap signals a renal cause.
⚠️ RTA is a normal-AG acidosis — a HIGH AG points elsewhere (ketoacidosis, toxins, lactate).
2. Type 1 (distal) — can't secrete H+
Distal (Type 1) RTA: the α-intercalated cell fails to secrete H+, so acid can't be excreted.
⚠️ Failure to acidify urine despite acidemia is the hallmark of distal RTA.
3. Type 1 — urine pH stays > 5.5
Because H+ can't be secreted, urine pH stays inappropriately high (> 5.5) even with systemic acidosis.
⚠️ An acidemic patient with urine pH > 5.5 = distal RTA (Type 2 can still acidify below 5.5).
4. Type 1 — HYPOkalemia
Distal RTA classically causes hypokalemia.
⚠️ Low K+ with normal-AG acidosis + alkaline urine = Type 1, not Type 4 (which is hyperkalemic).
5. Type 1 — stones & nephrocalcinosis
Alkaline urine + hypocitraturia + hypercalciuria → calcium-phosphate stones and nephrocalcinosis. Treat with alkali (bicarbonate / potassium citrate).
⚠️ Nephrocalcinosis / Ca-phosphate stones go with Type 1, not Type 2.
6. Type 1 — autoimmune (Sjögren)
Distal RTA is associated with autoimmune disease, especially Sjögren syndrome (also RA, lupus).
⚠️ A normal-AG acidosis in a Sjögren patient should prompt distal RTA.
7. Type 2 (proximal) — HCO3 wasting
Proximal (Type 2) RTA: the proximal tubule can't reabsorb filtered bicarbonate → bicarbonaturia.
⚠️ The defect is HCO3 reabsorption, not H+ secretion (that's Type 1).
8. Type 2 — Fanconi syndrome
Proximal RTA often comes with Fanconi: glucosuria (normal glucose), aminoaciduria, phosphaturia, uricosuria. Causes: myeloma, tenofovir, acetazolamide.
⚠️ Glucosuria with a NORMAL serum glucose + normal-AG acidosis = proximal RTA / Fanconi.
9. Type 2 — urine pH variable (<5.5)
Distal acidification is intact, so once serum HCO3 falls below the lowered threshold the urine CAN acidify (pH < 5.5).
⚠️ Ability to drop urine pH < 5.5 separates Type 2 from Type 1.
10. Type 2 — alkali (large) + K, thiazide
Needs LARGE doses of alkali plus potassium (giving HCO3 worsens K+ wasting); thiazides can help.
⚠️ Small bicarbonate doses fail in Type 2 — most spills back into the urine.
11. Type 4 — HYPERkalemia
Type 4 RTA is the only HYPERkalemic RTA — driven by hypoaldosteronism / aldosterone resistance.
⚠️ High K+ with a normal-AG acidosis = Type 4 (Types 1 & 2 are hypokalemic).
12. Type 4 — hypoaldosteronism & drug causes
Caused by low/ineffective aldosterone — diabetic hyporeninemic hypoaldosteronism, and drugs: ACEi/ARB, NSAIDs, trimethoprim, heparin, spironolactone.
⚠️ Trimethoprim and ACEi/ARB are classic exam triggers for Type 4.
13. Type 4 — acidic urine; treat fludrocortisone
Urine pH is usually < 5.5 (acidification intact); impaired ammoniagenesis limits acid excretion. Treat by lowering K+, low-K diet, and fludrocortisone if hypoaldosterone.
⚠️ Unlike Type 1, Type 4 still acidifies urine (pH < 5.5) — the problem is ammonium, not H+ pumps.
